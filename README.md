# TriCrypt-IoT

Lightweight Multi-Cipher Framework for Resource-Constrained IoT Devices

## Overview

TriCrypt-IoT is a lightweight encryption framework designed for IoT devices with limited memory and processing power.

The project introduces three custom encryption algorithms:

- PVRSC (Position-Variant Radial Stick Cipher)
- MSZC (Magic Square Zigzag Cipher)
- FSC (Fractal Substitution Cipher)
## Overview

TriCrypt-IoT is a lightweight encryption framework designed for IoT devices with limited memory and processing power.

The project introduces three custom encryption algorithms:

* **PVRSC (Position-Variant Radial Stick Cipher)**
* **MSZC (Magic Square Zigzag Cipher)**
* **FSC (Fractal Substitution Cipher)**

These algorithms were developed and tested on an ESP32 microcontroller to provide secure communication while maintaining low resource usage.

---

## Problem Statement

Many traditional encryption algorithms require significant computing resources, making them difficult to use on low-power IoT devices.

This project focuses on developing lightweight encryption techniques that improve security while reducing memory and processing requirements.
